Problem 3: define type coercion, provide 2 examples, explain each Type coersion happens when javascript encounters 2 things that are presented to it that are dissimilar, for example comparing two different types. Since Javascript doesn't like throwing errors, javascript casts (turns) things into other types in order to keep things running.

  • console.log(73+"Caitlin") is an example where trying to concatenate a number and string would result in a type error in python. But in javascript, it results in 73Caitlin.
  • console.log(1=="1") Normally in python this would return false since a number and a string don't have the same value. However, due to the casting in type coersion, it returns true in javascript
